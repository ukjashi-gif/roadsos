# 🚨 RoadSoS — Emergency Road Assistance Platform

RoadSoS provides location-based access to nearby trauma centres, ambulance services, vehicle rescue services, police stations, and emergency contacts during road accidents. The tool is designed to reduce emergency response time and improve access to life-saving assistance.

## 📌 Features

- 🚑 **One-Tap Emergency Calls** — Ambulance (108), Police (100), Fire (101), Highway Help (1033), Unified Emergency (112)
- 📍 **Location-Based Services** — Nearest trauma centres, police stations, blood banks, and vehicle rescue
- 🔧 **Vehicle Rescue Locator** — Find towing and recovery services near you
- 🩸 **Blood Bank Access** — Check availability and call instantly
- 🗺️ **Location Search** — Enter your area or use GPS to update nearby services
- 📋 **First Response Guide** — Step-by-step accident response checklist
- 📱 **Fully Responsive** — Works on mobile, tablet, and desktop

## 🚀 Getting Started

### Run Locally

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/roadsos.git
   ```
2. Open `index.html` in any browser — done.

### Deploy to GitHub Pages

1. Go to your repository → **Settings**
2. Navigate to **Pages → Source → main branch**
3. Save — your site will be live in a few seconds
